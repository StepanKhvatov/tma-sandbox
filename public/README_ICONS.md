# Иконки для Telegram Mini App

Для корректной работы приложения необходимо создать следующие иконки:

## Требуемые иконки:

1. **icon-192x192.png** - 192x192 пикселей (минимум для манифеста)
2. **icon-512x512.png** - 512x512 пикселей (для больших экранов)
3. **favicon.ico** - 32x32 пикселей (опционально)

## Рекомендации:

- Используйте PNG формат с прозрачностью
- Иконки должны быть maskable (работают с разными форматами)
- Используйте цветовую схему Telegram (#2481cc)
- Иконка должна быть узнаваемой на маленьком размере

## Инструменты для создания:

- [Figma](https://www.figma.com/) - для создания дизайна
- [ImageMagick](https://imagemagick.org/) - для конвертации размеров
- [RealFaviconGenerator](https://realfavicongenerator.net/) - онлайн генератор

## Временное решение:

До создания иконок можно использовать placeholder. Просто создайте простые цветные квадраты с логотипом вашего приложения.
